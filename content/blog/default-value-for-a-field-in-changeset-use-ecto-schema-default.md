Set a default value for an ecto schema like so: 

```elixir
defmodule Track.Time.Log do
  use Ecto.Schema
  import Ecto.Changeset

  schema "logs" do
    # ... other fields
    field(:billable, :boolean, default: true)
  end
end
```
